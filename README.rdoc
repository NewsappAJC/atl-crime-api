== AJC CRIME

Drop current table and set up new table for new crime data
     
    rake db:reset

Import new crime data

	rake import


API Routes available:

/crimes/:id
/crimes/current_year
/crimes/current_month
/crimes/month/:year/:month
/crimes/day/:year/:month/:day
/crimes/neighborhood/:neighborhood
/crimes/shift/:shift
/crimes/beat/:beat
